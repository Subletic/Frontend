Subletic-FAQ
Welche Dateiformate können für das Wörterbuch importiert werden?

Es können Json und Csv Dateien importiert werden.

Ist es möglich Wörterbücher verschiedener Sprachen zu importieren?

Nein, die Wörterbücher können nur in deutscher Sprache importiert werden.

Kann man mehrere Wörterbücher hinzufügen?

Ja, es ist möglich mehrere Wörterbücher hinzufügen.

Wie installiere ich Angular CLI?

Du kannst Angular CLI mit dem Befehl "npm install -g @angular/cli" installieren.
